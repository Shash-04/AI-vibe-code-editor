## 🚀 Getting Started

### 1. Installation

Run the following command to install dependencies:

```bash
npm install
```

### 2. Development

Start the development server:

```bash
npm run dev
```

Your app will be available at `http://localhost:3000`.

### 3. Build

Prepare for production:

```bash
npm run build
```
